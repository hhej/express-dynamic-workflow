  <behavior>
    The surcharge formula (from docs/architecture.md):

    1. fuel_delta_pct = (current_diesel_price - BASELINE_DIESEL_PRICE) / BASELINE_DIESEL_PRICE
    2. surcharge_pct = fuel_delta_pct * SHIPPING_MULTIPLIERS[shipping_type]
    3. If shipping_type == "bounce": surcharge_pct += traffic_severity * 0.02
    4. Clamp surcharge_pct to [SURCHARGE_FLOOR, SURCHARGE_CAP] (i.e., [-0.05, 0.15])
    5. surcharge_amount = base_rate * surcharge_pct
    6. total = base_rate + surcharge_amount
    7. Return SurchargeResult with rounded values (4 decimal places for pct, 2 for amounts)

    Hand-calculated test cases (per D-12):

    **Case 1: Bounce at baseline price, no traffic (severity=1)**
    - base_rate=100, current_diesel=29.94, shipping_type="bounce", traffic_severity=1
    - fuel_delta_pct = (29.94 - 29.94) / 29.94 = 0.0
    - surcharge_pct = 0.0 * 1.0 = 0.0
    - traffic adj: 0.0 + 1 * 0.02 = 0.02
    - No cap/floor hit
    - surcharge_amount = 100 * 0.02 = 2.0
    - total = 102.0
    - Expected: pct=0.02, amount=2.0, total=102.0, capped=False

    **Case 2: Retail Standard, price above baseline**
    - base_rate=200, current_diesel=32.94, shipping_type="retail_standard", traffic_severity=3
    - fuel_delta_pct = (32.94 - 29.94) / 29.94 = 0.10020... (~0.1002)
    - surcharge_pct = 0.10020 * 0.5 = 0.05010...
    - NO traffic adj (not bounce)
    - surcharge_amount = 200 * 0.0501 = 10.02
    - total = 210.02
    - Expected: pct~0.0501, amount~10.02, total~210.02, capped=False

    **Case 3: Retail Fast, price below baseline**
    - base_rate=150, current_diesel=28.00, shipping_type="retail_fast", traffic_severity=1
    - fuel_delta_pct = (28.00 - 29.94) / 29.94 = -0.06479... (~-0.0648)
    - surcharge_pct = -0.0648 * 0.8 = -0.05183...
    - Floor hit: clamp to -0.05
    - surcharge_amount = 150 * -0.05 = -7.50
    - total = 142.50
    - Expected: pct=-0.05, amount=-7.50, total=142.50, capped=True

    - base_rate=100, current_diesel=33.00, shipping_type="bounce", traffic_severity=5
    - fuel_delta_pct = (33.00 - 29.94) / 29.94 = 0.10220... (~0.1022)
    - surcharge_pct = 0.1022 * 1.0 = 0.1022
